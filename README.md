# Unit Converter Web App

A simple web-based unit converter built with HTML, CSS, and JavaScript. This application allows users to convert values between multiple unit categories such as mass, length, and temperature quickly and easily through a user-friendly interface.

## Features

- Convert between multiple unit categories:
  - Mass (mg, g, kg)
  - Length (mm, cm, m, inch, ft, yd)
  - Temperature (Celsius, Kelvin, Fahrenheit)
- Instant conversion results
- User-friendly web interface
- Handles invalid input gracefully
- Scalable structure to support additional unit types

## Technologies Used

- HTML for structure
- CSS for styling and layout
- JavaScript for conversion logic and interactivity

## How to Run

1. Clone the repository:

git clone https://github.com/your-username/unit-converter-web.git

2. Navigate into the project folder:

cd unit-converter-web

3. Open the `index.html` file in your web browser to start using the converter.

## Example Usage

1. Enter the value to convert
2. Select the “from” unit (e.g., mm, mg, K)
3. Select the “to” unit (e.g., cm, kg, F)
4. Conversion result is displayed instantly on the page
